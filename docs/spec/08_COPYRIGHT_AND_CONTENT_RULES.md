# 内容与版权规则

## 1. 可以使用

- 自己编写的知识总结
- 自己编写的题目和解析
- 官方公开且允许使用的信息摘要
- 用户购买教材后自己整理的个人笔记
- 获得明确授权的题目
- AI生成后经过人工审核的原创练习草稿

## 2. 不可以直接做

- 扫描并上传整本教材
- 复制整套商业题库
- 抓取付费网站题目
- 将培训机构解析批量改几个词后当原创
- 公开发布来源不明的“历年真题全集”
- 把AI生成内容标记为官方真题

## 3. 内容元数据

每条知识点和题目必须记录：

- source_type
- source_note
- copyright_status
- review_status
- version
- created_at
- updated_at

## 4. AI内容

AI生成的：

- 解析
- 变式题
- 章节总结

默认状态为 `ai_draft` 和 `pending_review`。

必须人工确认：

- 事实正确
- 答案唯一或评分规则合理
- 不与已知商业题目高度重复
- 无虚假来源
- 无错误法律条文

确认后才可改为 `published`。

## 5. 纠错

用户可以对题目提交：

- 答案争议
- 解析错误
- 题干不完整
- 选项重复
- 知识已过期
- 版权疑问

存在重大争议的题目标记 `disputed` 并从随机练习和模考中排除，直到审核完成。
