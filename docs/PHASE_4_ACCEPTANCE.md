# Phase 4 验收记录

日期：2026-07-18

## 已完成

- 答错后自动保留错题历史，复习等级重置为零，并按上海自然日在次日零点
  到期。
- 复习间隔固定为1/3/7/14/30/60天；答对但不确定提升一级，答对且确定
  提升两级，最高不超过五级。
- 连续两次答对后标记暂时掌握，但保留错题和作答历史；再次答错会重新
  激活并重置复习进度。
- 九类错因、手动标记掌握、今日到期、逾期和指定错题立即再练均可用。
- 收藏和错题状态相互独立，手动掌握不会取消收藏。
- 错题列表支持状态、模块、错因、至少错误次数和最近错误时间筛选，所有
  学习状态继续按 `user_id` 隔离。
- 手机端错题本展示顶部统计、复习等级、错因、下次复习时间、收藏状态、
  到期队列入口和单题再练入口。
- Supabase迁移扩展用户题目状态与作答错因，回填既有错题，新增复习会话、
  错因和掌握RPC。
- Phase 4表字段继续使用既有RLS边界；新增RPC撤销 `public`、`anon` 和
  `authenticated` 权限，仅授予 `service_role`。
