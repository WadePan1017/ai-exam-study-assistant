# Phase 3 验收记录

日期：2026-07-18

## 已完成

- 新增单选、多选和判断题的导入格式、来源与版权门禁、危险内容检查、
  知识点关联检查和递增版本规则。
- 题目导入先校验和预览，再显式确认；支持新增、新版本、跳过与拒绝，
  存在拒绝项时不写入同一文件中的其他题目。
- 顺序、章节、随机和只练未做题四种模式可用，并展示当前真实可用题量。
- 章节模式只抽取选定一级模块下的题目；随机练习同一会话不重复。
- 单选、判断精确匹配；多选忽略选择顺序并采用全对才得分。
- 未提交前不返回正确答案、解析和历史作答；提交后立即反馈并展开解析。
- 支持上一题、下一题、题号导航、收藏和“不确定”标记。
- 会话保存题目顺序、题目版本、当前位置、已答题号和结果，刷新后可恢复。
- 每次作答独立追加历史记录；题目新版本不会改变旧会话和旧作答版本。
- 收藏、会话和作答历史按 `user_id` 隔离。
- Supabase迁移新增题目版本、选项、知识点关联、练习会话、会话题目、
  作答历史和用户题目状态。
- 题目导入、创建会话和提交答案由仅限 `service_role` 的RPC执行；所有
  Phase 3表启用RLS并撤销浏览器角色权限。
- 本地种子和E2E题目均为原创示例，不包含第三方仓库PDF或商业题库内容。
