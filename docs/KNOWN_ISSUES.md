# 已知问题

## Phase 1至Phase 5

- 尚未创建和连接实际 Supabase 云端项目，因此迁移、限流 RPC 和 RLS
  仍未在干净云数据库上执行验收；知识点与题目导入、练习会话和作答RPC
  、错题复习、模考会话和评分RPC同样未做云端验收。
- 本地未配置Supabase时，知识点导入结果和个人笔记只保存在开发服务器
  进程内；题目、会话、收藏和作答历史也会在服务器重启后恢复为内置原创
  示例；错题、复习进度和模考会话也不会跨服务器重启保留。断网答案草稿
  保留在当前浏览器，但不能代替正式数据库备份。
- 当前本地2题模考只用于验证软件流程，不构成完整备考题库或官方模拟卷。
  学习报告和AI功能属于后续阶段。
- Phase 5只对可确定判分的单选、多选和判断题自动评分；主观案例题保存、
  人工自评和AI辅助尚未实现。
- 当前只提供基础离线友好布局，PWA安装与离线缓存属于Phase 8。
- 正式部署地址和手机真机验收需要在获得云端环境配置后完成。
- 退出只清除当前设备 Cookie；如会话 Cookie 疑似泄露，需要轮换
  `APP_SESSION_SECRET` 撤销所有旧会话。
